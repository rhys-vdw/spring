# alias Facing
---



```lua
(alias) Facing = (0|1|2|3|"s"|"e"|"n"|"w"|"south"|"east"...)
    | 0 -- South
    | 1 -- East
    | 2 -- North
    | 3 -- West
    | "s" -- South
    | "e" -- East
    | "n" -- North
    | "w" -- West
    | "south" -- South
    | "east" -- East
    | "north" -- North
    | "west" -- West

```




Facing direction represented by a string or number.
